During my internship at **Amazon Web Services Viet Nam Company Limited** from **17/04/2026 to 09/07/2026**, I had the opportunity to learn, practice, and apply cloud computing knowledge in a structured internship environment through the **Workforce Bootcamp - First Cloud AI Journey** program.

In the first seven weeks, I focused on learning AWS fundamental services through AWS Study Group workshops and hands-on labs. The main topics included AWS Free Tier, AWS CLI, IAM, VPC, EC2, RDS, AWS Backup, CloudWatch, AWS Budgets, Docker, ECS, CI/CD, Security Hub, Route 53 Resolver, and Transit Gateway.

From week 8 to week 12, I worked on the final project named **PharmaCare AI**, an online pharmacy management and medicine-selling website integrated with an AI chatbot. Through this project, I analyzed requirements, designed the AWS architecture, selected suitable AWS services, prepared the database design, and documented the deployment workflow. The project helped me understand how AWS services can be combined to build a practical cloud-based system.

Through the internship, I improved my skills in cloud computing, AWS service usage, system analysis, architecture design, technical documentation, problem-solving, and self-learning. I also learned how to organize internship worklogs, write technical blogs, prepare project proposals, and document workshop implementation steps.

In terms of work ethic, I tried to complete assigned tasks responsibly, followed the internship timeline, actively studied the provided materials, and improved my knowledge through hands-on practice. Although I still need to improve some skills, this internship helped me build a stronger foundation for future work in cloud computing and information security.

### Needs Improvement

* Strengthen practical experience with advanced AWS services such as Amazon Bedrock, Amazon OpenSearch Service, Amazon Cognito, API Gateway, and AWS Lambda in real deployment scenarios.

* Improve troubleshooting skills when working with cloud infrastructure, especially in networking, authentication, permissions, and serverless integration.

* Enhance communication and presentation skills to explain technical architecture more clearly and confidently.

* Improve English technical writing skills to make reports, blogs, and documentation more professional.

* Continue practicing cost management and resource cleanup to avoid unnecessary AWS charges during hands-on labs.

* Strengthen time management skills when working on multiple report sections such as Worklog, Proposal, Project, Workshop, Blogs, and Self-Assessment.

### Overall Reflection

Overall, I believe that I completed the internship with a positive learning attitude and achieved the main objectives of the program. I gained practical knowledge of core AWS services, understood the basic process of designing cloud architecture, and applied the knowledge to the PharmaCare AI project.

The internship helped me realize that cloud computing is not only about using individual AWS services, but also about designing secure, reliable, scalable, cost-optimized, and well-documented systems. This experience gave me a clearer direction for improving my technical skills and preparing for future career opportunities in cloud computing and information security.
