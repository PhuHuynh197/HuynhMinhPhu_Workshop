This section presents my internship worklog during the **Workforce Bootcamp - First Cloud AI Journey** program at **Amazon Web Services Viet Nam Company Limited**.

The internship lasted for **12 weeks**, from **17/04/2026 to 09/07/2026**. During the first seven weeks, I focused on learning AWS fundamental services through AWS Study Group workshops and hands-on labs. From week 8 to week 12, I worked on the final project named **PharmaCare AI**, a pharmacy management and online medicine-selling website integrated with an AI chatbot on AWS.

### Internship Timeline

**Week 1:** [Getting familiar with the internship program, AWS account, AWS Console, AWS CLI, and basic cloud concepts](1.1-week1/)

**Week 2:** [Learning AWS Identity and Access Management, IAM users, groups, policies, roles, and permission control](1.2-week2/)

**Week 3:** [Learning Amazon VPC, subnets, route tables, Internet Gateway, NAT Gateway, Security Groups, and network security](1.3-week3/)

**Week 4:** [Learning Amazon EC2, Linux and Windows instances, key pairs, SSH, RDP, EBS, Snapshot, and AMI](1.4-week4/)

**Week 5:** [Learning Amazon RDS, DB Subnet Group, database security, backup, restore, and application database connection](1.5-week5/)

**Week 6:** [Learning Load Balancer, Auto Scaling, Docker, container deployment, and basic application scalability on AWS](1.6-week6/)

**Week 7:** [Learning AWS Budget, CloudWatch, AWS Backup, Route 53, security monitoring, and other advanced AWS services](1.7-week7/)

**Week 8:** [Analyzing requirements and designing the AWS architecture for the PharmaCare AI project](1.8-week8/)

**Week 9:** [Designing the database and building the network infrastructure with VPC, subnets, security groups, and Amazon RDS](1.9-week9/)

**Week 10:** [Deploying the frontend, authentication, API layer, and backend services using S3, CloudFront, Cognito, API Gateway, and Lambda](1.10-week10/)

**Week 11:** [Building the AI chatbot and RAG architecture using Amazon OpenSearch Service, Amazon Bedrock, Amazon S3, and Lambda](1.11-week11/)

**Week 12:** [Testing, monitoring, backup configuration, project finalization, workshop documentation, and internship report completion](1.12-week12/)
