Phần này trình bày nhật ký thực tập của em trong chương trình **Workforce Bootcamp - First Cloud AI Journey** tại **Công ty TNHH Amazon Web Services Viet Nam**.

Thời gian thực tập kéo dài **12 tuần**, từ ngày **17/04/2026 đến ngày 09/07/2026**. Trong 7 tuần đầu tiên, em tập trung tìm hiểu các dịch vụ nền tảng của AWS thông qua các bài workshop và bài thực hành trên AWS Study Group. Từ tuần 8 đến tuần 12, em thực hiện dự án cuối kỳ với đề tài **PharmaCare AI**, một website quản lý nhà thuốc, bán thuốc online và tích hợp chatbot AI tư vấn khách hàng trên nền tảng AWS.

### Tiến độ thực tập

**Tuần 1:** [Làm quen chương trình thực tập, tài khoản AWS, AWS Console, AWS CLI và các khái niệm Cloud cơ bản](1.1-week1/)

**Tuần 2:** [Tìm hiểu AWS Identity and Access Management, IAM User, Group, Policy, Role và phân quyền truy cập](1.2-week2/)

**Tuần 3:** [Tìm hiểu Amazon VPC, Subnet, Route Table, Internet Gateway, NAT Gateway, Security Group và bảo mật mạng](1.3-week3/)

**Tuần 4:** [Tìm hiểu Amazon EC2, máy chủ Linux và Windows, Key Pair, SSH, RDP, EBS, Snapshot và AMI](1.4-week4/)

**Tuần 5:** [Tìm hiểu Amazon RDS, DB Subnet Group, bảo mật cơ sở dữ liệu, sao lưu, khôi phục và kết nối ứng dụng với database](1.5-week5/)

**Tuần 6:** [Tìm hiểu Load Balancer, Auto Scaling, Docker, triển khai container và khả năng mở rộng ứng dụng trên AWS](1.6-week6/)

**Tuần 7:** [Tìm hiểu AWS Budget, CloudWatch, AWS Backup, Route 53, giám sát bảo mật và các dịch vụ AWS nâng cao khác](1.7-week7/)

**Tuần 8:** [Phân tích yêu cầu và thiết kế kiến trúc AWS cho dự án PharmaCare AI](1.8-week8/)

**Tuần 9:** [Thiết kế cơ sở dữ liệu và xây dựng hạ tầng mạng với VPC, Subnet, Security Group và Amazon RDS](1.9-week9/)

**Tuần 10:** [Triển khai frontend, xác thực người dùng, API layer và backend bằng S3, CloudFront, Cognito, API Gateway và Lambda](1.10-week10/)

**Tuần 11:** [Xây dựng chatbot AI và kiến trúc RAG bằng Amazon OpenSearch Service, Amazon Bedrock, Amazon S3 và Lambda](1.11-week11/)

**Tuần 12:** [Kiểm thử, giám sát, cấu hình backup, hoàn thiện dự án, viết workshop và hoàn tất báo cáo thực tập](1.12-week12/)
